
# Proyecto Final - JEGM 3 WEB A

Este repositorio contiene mi primer proyecto de desarrollo web construido con HTML y CSS básico. Este trabajo fue desarrollado como parte del programa de DEV.F, bajo la guía del instructor Daniel Gloria.

## Requisitos Previos

Para visualizar y editar este proyecto en tu entorno local, te sugiero tener instalado lo siguiente:
* [Visual Studio Code (VS Code)](https://code.visualstudio.com/)
* La extensión **Live Server** para VS Code.
* *(Opcional)* [Git](https://git-scm.com/) si prefieres clonar el repositorio desde la terminal.

## Cómo ver la página en local

Sigue estos pasos para descargar y ejecutar el proyecto en tu computadora:

### 1. Descargar el repositorio

**Opción A: Usando Git (Recomendado)**
Abre tu terminal y ejecuta el siguiente comando para clonar el proyecto:
